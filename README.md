# 리마켓 프로젝트
2022.08. - 2022.11.

과천의 오래된 ‘굴다리 시장’에 대해 시장 상인들의 인터뷰와 개성적인 점포의 특징을 담은 가이드북을 만들어 시민들에게 전달하는 한편, 오랫동안 자리를 지켜온 시장에 대한 시민들의 인상을 글, 사진, 그림의 형태로 함께 나눔으로써 시장을 새롭게 변화시키는 프로젝트.
